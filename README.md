# B2B Supply Chain & Sales Analysis (SQL & Power BI)

## 📌 Project Overview
This project focuses on analyzing supply chain performance and sales data for a B2B business. Using SQL, I extracted actionable insights regarding order management, shipping efficiency, and sales performance. Additionally, a Power BI dashboard was utilized to visualize key performance indicators (KPIs) to help stakeholders optimize inventory and logistics.

## 🛠️ Tech Stack Used
* **Database/Querying:** MySQL / SQL
* **Data Visualization:** Power BI
* **Documentation:** GitHub Markdown

## 🔍 Key Data Insights Extracted (SQL)
* **Sales Performance:** Identified top-performing product categories and high-value B2B customers.
* **Shipping Efficiency:** Calculated average shipping delays and identified regions with maximum transit times.
* **Inventory Optimization:** Analyzed stock rotation and order frequency to minimize supply chain bottlenecks.
